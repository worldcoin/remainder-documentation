Multilinear Extensions (MLEs)

Let f(x1​,x2​,…,xn​) be a function ∈F[X1​,...,Xn​]<2. Its multilinear extension f​(x1​,…,xn​) is defined such that f​ is linear in each xj​ and xj​∈F, such that f​(x)=f(x)∀x∈{0,1}n.

Equality MLE

In order to explicitly formulate f​ in terms of f, let us define the following indicator function:

eq​(x;z):{0,1}2n↦{0,1}={1,0,​if x=zotherwise.​

Then, we can see that if eq(x,z) were linear in the x variables, i.e., we could define a multilinear extension for eq, called eq​, we can define f​(x1​,…,xn​)=zi​∈{0,1}∑​eq​(x;z)⋅f(z1​,…,zn​) where zi​ are the bits of z.

Fortunately, eq(x;z):{0,1}2n→{0,1} has an explicit formula which is linear in each of xi​, or the bits of x. Intuitively, if x=z, then each of its bits must be equal. In boolean logic, this is the same thing as saying (xi​=zi​=0) OR (xi​=zi​=1) for all of the bits i (which is an AND over all of the bits i).

When our inputs xi​,zi​∈{0,1} this statement can be expressed as the following product: i=1∏n​(1−xi​)(1−zi​)+xi​zi​. Taking the multilinear extension of eq simply means allowing for non-binary inputs xi​,zi​∈F, because the polynomial is already linear in each variable. Because when xi​,zi​∈{0,1}, eq(x;z)=eq​(x;z), we have the multilinear extension eq​(x;z).

Another property of multilinear extensions is that they are uniquely defined. I.e., ∑zi​∈{0,1}​eq​(x;z)⋅f(z1​,…,zn​) is the only multilinear function in n variables which extends f.

Example

Let f(x1​,x2​,x3​)=2x12​x3​+4x2​x33​+3x1​x22​x3​+5x1​+6x2​+3. Let us first build a table of evaluations of f for xi​∈{0,1}:

(x1​,x2​,x3​)f(x1​,x2​,x3​)
(0,0,0)3
(0,0,1)3
(0,1,0)9
(0,1,1)13
(1,0,0)8
(1,0,1)10
(1,1,0)14
(1,1,1)23

We also build a table for eq​(x;z) for xi​∈{0,1} in terms of z:

(z1​,z2​,z3​)eq​(x;z)
(0,0,0)(1−x1​)(1−x2​)(1−x3​)
(0,0,1)(1−x1​)(1−x2​)(x3​)
(0,1,0)(1−x1​)(x2​)(1−x3​)
(0,1,1)(1−x1​)(x2​)(x3​)
(1,0,0)(x1​)(1−x2​)(1−x3​)
(1,0,1)(x1​)(1−x2​)(x3​)
(1,1,0)(x1​)(x2​)(1−x3​)
(1,1,1)(x1​)(x2​)(x3​)

Then, using the formula for f​(x1​,…,xn​)=∑zi​∈{0,1}​eq​(x;z)⋅f(z1​,…,zn​), we get the explicit formula:

f​(x1​,…,xn​)=3(1−x1​)(1−x2​)(1−x3​)+3(1−x1​)(1−x2​)(x3​)+9(1−x1​)(x2​)(1−x3​)+13(1−x1​)(x2​)(x3​)+8(x1​)(1−x2​)(1−x3​)+10(x1​)(1−x2​)(x3​)+14(x1​)(x2​)(1−x3​)+23(x1​)(x2​)(x3​).

From here you can verify that f​(x)=f(x) when x∈{0,1}n, and that f​(x) is linear in each of the x variables.
